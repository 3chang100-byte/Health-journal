# 헬스 저널 PWA

체중·식사·공복을 한눈에 보는 건강 트래커. PWA로 만들어져서 휴대폰 홈 화면에 추가하면 일반 앱처럼 전체화면으로 열려요.

## 🚀 가장 쉬운 배포 방법: Vercel (5분 완료)

### 1단계: 압축 파일 풀기
다운로드한 `health-pwa.zip`을 압축 해제하세요.

### 2단계: Vercel 가입 (무료)
1. [vercel.com](https://vercel.com) 접속
2. **Sign Up** → GitHub 또는 이메일로 가입

### 3단계: 프로젝트 업로드
**가장 쉬운 방법 — 드래그앤드롭:**
1. Vercel 대시보드에서 **Add New** → **Project** 클릭
2. 화면 어디서든 **압축 해제한 폴더 전체**를 브라우저에 드래그앤드롭
3. 프로젝트 이름 정하기 (예: `health-journal`)
4. **Deploy** 클릭
5. 30초~1분 후 배포 완료!

배포가 끝나면 `https://health-journal-xxx.vercel.app` 같은 URL이 생겨요.

### 4단계: 휴대폰 홈 화면에 추가

**📱 iPhone (Safari):**
1. Safari로 배포된 URL 열기
2. 하단 **공유 버튼** (□↑) 탭
3. **홈 화면에 추가** 선택
4. 이름 확인 후 **추가**

**📱 Android (Chrome):**
1. Chrome으로 배포된 URL 열기
2. 우상단 **⋮ 메뉴** 탭
3. **앱 설치** 또는 **홈 화면에 추가** 선택
4. **설치** 탭

이제 홈 화면에 헬스저널 아이콘이 생기고, 탭하면 일반 앱처럼 전체화면으로 열려요!

## 🔑 사진 자동 분석 사용하려면

앱 안에서 우상단 **🔑 아이콘**을 탭하고 Anthropic API 키를 입력하세요.

1. [console.anthropic.com](https://console.anthropic.com/settings/keys)에서 가입
2. **API Keys** → **Create Key**
3. `sk-ant-api03-...`로 시작하는 키를 복사
4. 앱의 키 설정 화면에 붙여넣기

> 키는 **이 휴대폰 브라우저에만** 저장되며, 사진 분석할 때만 Anthropic 서버로 직접 전송돼요. 다른 곳으로는 절대 가지 않아요.

> 사용량은 사진 한 장당 약 $0.01~0.02 정도예요.

---

## 💾 데이터는 어디에 저장되나요?

휴대폰 브라우저의 **localStorage**에 저장돼요. 이 말은:

✅ **유지되는 경우**
- 앱을 닫았다가 다시 열어도 그대로
- 휴대폰을 재부팅해도 그대로
- 코드를 업데이트하고 재배포해도 그대로 (key가 같아서)

⚠️ **사라질 수 있는 경우**
- 브라우저 데이터를 수동으로 삭제할 때
- 휴대폰을 바꿀 때
- 시크릿 모드로 접속할 때

**그래서 가끔 백업하세요!** 앱 안의 **백업** 탭에서 JSON 파일로 내보낼 수 있고, 새 휴대폰에서 다시 불러올 수 있어요.

## 🔧 로컬에서 테스트하려면

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 열기.

빌드:
```bash
npm run build
```

`dist/` 폴더가 생기고, 이걸 어디든 정적 호스팅에 올리면 돼요.

## 🆘 문제가 생기면

**"홈 화면에 추가 메뉴가 안 보여요"**
- iOS는 **반드시 Safari** 사용 (Chrome iOS는 안 됨)
- Android Chrome에서 PWA를 인식하려면 사이트를 한 번 사용한 뒤 메뉴가 나타날 때까지 잠시 기다려주세요

**"오프라인에서 안 열려요"**
- 최초 1번은 인터넷 연결 상태에서 앱을 켜야 캐시가 생성돼요. 그 다음부터는 오프라인에서도 열려요.

**"사진 분석이 안 돼요"**
- API 키가 올바르게 입력됐는지 확인
- 인터넷 연결 확인
- API 사용량 한도(크레딧) 확인
